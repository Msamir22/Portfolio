Yard Management System

Capstone Logistics 2021

Real-time RFID trailer tracking system integrated with the Managed Receiving platform. Built reusable Angular components for complex data visualization and live tracking.

Project Overview

A yard management solution that uses RFID technology to track trailers in real-time across warehouse yards. The system integrates with the Managed Receiving platform to provide seamless logistics operations.

My Role

  • Implemented real-time tracking UI with RxJS
  • Built interactive yard map component
  • Created integration layer with MR platform
  • Optimized performance for large datasets
